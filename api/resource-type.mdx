The `ResourceType` enum identifies the type of a compiled Source 2 resource file. Each resource type corresponds to a specific file extension and contains different data structures.

## Usage Examples

### Determining Resource Type

```csharp
using ValveResourceFormat;

using var resource = new Resource();
resource.Read("materials/metal.vmat_c");

switch (resource.ResourceType)
{
    case ResourceType.Material:
        Console.WriteLine("This is a material file");
        break;
    case ResourceType.Texture:
        Console.WriteLine("This is a texture file");
        break;
    case ResourceType.Model:
        Console.WriteLine("This is a model file");
        break;
    default:
        Console.WriteLine($"Unknown or unsupported type: {resource.ResourceType}");
        break;
}
```

### Filtering Resources by Type

```csharp
var resourceFiles = Directory.GetFiles("extracted", "*_c", SearchOption.AllDirectories);
var materials = new List<string>();

foreach (var file in resourceFiles)
{
    using var resource = new Resource();
    try
    {
        resource.Read(file);
        if (resource.ResourceType == ResourceType.Material)
        {
            materials.Add(file);
        }
    }
    catch (Exception ex)
    {
        Console.WriteLine($"Failed to read {file}: {ex.Message}");
    }
}

Console.WriteLine($"Found {materials.Count} material files");
```

### Type-Specific Processing

```csharp
using var resource = new Resource();
resource.Read("example_c");

if (resource.ResourceType == ResourceType.Model)
{
    var model = (Model)resource.DataBlock;
    // Process model data
}
else if (resource.ResourceType == ResourceType.Texture)
{
    var texture = (Texture)resource.DataBlock;
    // Process texture data
}
else if (resource.ResourceType == ResourceType.Sound)
{
    var sound = (Sound)resource.DataBlock;
    // Process sound data
}
```

## Extension Mapping

Each ResourceType has an associated file extension attribute. Use `ResourceTypeExtensions.DetermineByFileExtension()` to determine the type from a file extension:

```csharp
var extension = Path.GetExtension("materials/example.vmat_c");
var resourceType = ResourceTypeExtensions.DetermineByFileExtension(extension);
// Returns: ResourceType.Material
```
