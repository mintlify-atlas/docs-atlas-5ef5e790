Get up and running with ValveResourceFormat by following this quick guide. You'll learn how to install the library, open a VPK archive, read a resource file, and export it.

## Installation

<Steps>
  <Step title="Install the package">
    Install ValveResourceFormat via NuGet:
    
    <CodeGroup>
    ```bash .NET CLI
    dotnet add package ValveResourceFormat
    dotnet add package ValvePak
    ```
    
    ```bash Package Manager
    Install-Package ValveResourceFormat
    Install-Package ValvePak
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Add using statements">
    Add the required namespaces to your C# file:
    
    ```csharp
    using ValveResourceFormat;
    using ValveResourceFormat.ResourceTypes;
    using ValveResourceFormat.IO;
    using SteamDatabase.ValvePak;
    ```
  </Step>
</Steps>

## Basic Example

Here's a complete example showing how to open a VPK archive, read a texture, and export it to PNG:

<Steps>
  <Step title="Open a VPK package">
    ```csharp
    // Open the VPK archive
    using var package = new Package();
    package.Read("pak01_dir.vpk");
    
    // Find a texture file
    var entry = package.FindEntry("materials/models/props/debug.vtex_c");
    if (entry == null)
    {
        Console.WriteLine("File not found");
        return;
    }
    
    // Read the file data
    package.ReadEntry(entry, out var fileData);
    ```
  </Step>
  
  <Step title="Parse the resource file">
    ```csharp
    // Create and read the resource
    using var resource = new Resource();
    resource.FileName = entry.GetFullPath();
    
    using var stream = new MemoryStream(fileData);
    resource.Read(stream);
    
    // Verify it's a texture
    if (resource.ResourceType != ResourceType.Texture)
    {
        Console.WriteLine("Not a texture file");
        return;
    }
    ```
  </Step>
  
  <Step title="Export to PNG">
    ```csharp
    // Get the texture data
    var texture = (Texture)resource.DataBlock;
    
    // Generate a bitmap
    using var bitmap = texture.GenerateBitmap();
    
    // Convert to PNG
    var png = TextureExtract.ToPngImage(bitmap);
    
    // Save to disk
    File.WriteAllBytes("output.png", png);
    Console.WriteLine("Texture exported successfully!");
    ```
  </Step>
</Steps>

## Working with Different Resource Types

ValveResourceFormat supports many resource types. Here are quick examples for common ones:

<Tabs>
  <Tab title="Models">
    ```csharp
    using var resource = new Resource();
    resource.Read("character.vmdl_c");
    
    if (resource.ResourceType == ResourceType.Model)
    {
        var model = (Model)resource.DataBlock;
        Console.WriteLine($"Model: {model.Name}");
        Console.WriteLine($"Meshes: {model.GetEmbeddedMeshes().Count()}");
    }
    ```
  </Tab>
  
  <Tab title="Materials">
    ```csharp
    using var resource = new Resource();
    resource.Read("metal.vmat_c");
    
    if (resource.ResourceType == ResourceType.Material)
    {
        var material = (Material)resource.DataBlock;
        Console.WriteLine($"Shader: {material.ShaderName}");
        
        foreach (var texture in material.TextureParams)
        {
            Console.WriteLine($"{texture.Key}: {texture.Value}");
        }
    }
    ```
  </Tab>
  
  <Tab title="Worlds">
    ```csharp
    using var resource = new Resource();
    resource.Read("de_dust2.vwrld_c");
    
    if (resource.ResourceType == ResourceType.World)
    {
        var world = (World)resource.DataBlock;
        var entityLumps = world.GetEntityLumpNames();
        Console.WriteLine($"Entity lumps: {string.Join(", ", entityLumps)}");
    }
    ```
  </Tab>
  
  <Tab title="Sounds">
    ```csharp
    using var resource = new Resource();
    resource.Read("music.vsnd_c");
    
    if (resource.ResourceType == ResourceType.Sound)
    {
        var sound = (Sound)resource.DataBlock;
        Console.WriteLine($"Format: {sound.Type}");
        Console.WriteLine($"Sample Rate: {sound.SampleRate}");
        Console.WriteLine($"Channels: {sound.ChannelCount}");
        
        var audioData = sound.GetSound();
        File.WriteAllBytes("output.wav", audioData);
    }
    ```
  </Tab>
</Tabs>

## Using FileExtract Helper

For automatic format detection and extraction, use the `FileExtract` helper:

```csharp
using var resource = new Resource();
resource.Read("some_file.vtex_c");

// Automatically extract based on file type
var contentFile = FileExtract.Extract(resource);

// Write to disk
using var output = File.OpenWrite("output" + contentFile.Extension);
output.Write(contentFile.Data);
```

## Common Patterns

### Error Handling

```csharp
try
{
    using var resource = new Resource();
    resource.Read(filePath);
    
    // Process resource...
}
catch (InvalidDataException ex)
{
    Console.WriteLine($"Invalid resource file: {ex.Message}");
}
catch (IOException ex)
{
    Console.WriteLine($"File error: {ex.Message}");
}
```

### Batch Processing

```csharp
var files = Directory.GetFiles(".", "*.vtex_c");

foreach (var file in files)
{
    using var resource = new Resource();
    resource.Read(file);
    
    if (resource.ResourceType == ResourceType.Texture)
    {
        var texture = (Texture)resource.DataBlock;
        using var bitmap = texture.GenerateBitmap();
        var png = TextureExtract.ToPngImage(bitmap);
        
        var outputName = Path.GetFileNameWithoutExtension(file) + ".png";
        File.WriteAllBytes(outputName, png);
    }
}
```

### Resource Disposal

Always dispose of resources properly:

```csharp
// Using statement (recommended)
using var resource = new Resource();
resource.Read(filePath);

// Manual disposal
Resource resource = null;
try
{
    resource = new Resource();
    resource.Read(filePath);
    // Use resource...
}
finally
{
    resource?.Dispose();
}
```
